# LibcurlHttp

# Quickstart/Demo
```CPP
LibcurlHttp* http = HTTP_CLIENT::Ins().CreateHttp();
HTTP_CLIENT::Ins().setProgress(http, PROGRESS_CALLBACK, NULL);
HTTP_CLIENT::Ins().download(http, "https://sm.myapp.com/original/Download/LeapFTPSetup_3.1.0.50.exe", "E:\\my\\app");
```
